ScrollUp 3

ScrollUp is a lightweight JavaScript plugin to create a customisable "Scroll to top" widget that will work with any website, with ease.

Download on GitHub →

Why use ScrollUp?

With many modern websites, page lengths are growing larger and larger—especailly parallax based websites. ScrollUp provides an easy way back up for your users.

Demo themes

As of v3, ScrollUp doesn't inject any markup or styles, allowing full control how it appears without needing to configure a bunch of options. Add in animate.css to create some slick animations, or even create your own.

Example themes:

  • Tab
  • Font Icon
  • Link
  • Image
Demo themes can be found in the css/themes/ folder.

Browser Support

ScrollUp uses modern API's and requires IE10+. If you need to support a lower version of IE, shims are available:

  • ClassList API
  • requestAnimationFrame

Requirements

ScrollUp is dependency free, no requirements necessary.
